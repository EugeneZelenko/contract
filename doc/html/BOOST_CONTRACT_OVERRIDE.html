Macro BOOST_CONTRACT_OVERRIDE

BOOST_CONTRACT_OVERRIDE — Declare an override type naming it override_function_name.

Synopsis

// In header: <boost/contract/override.hpp>

BOOST_CONTRACT_OVERRIDE(function_name)

Description

Declare the override type to be passed as an explicit template parameter to boost::contract::public_function for overriding public functions.

See Also:

Tutorial

Parameters:

function_name

Name of the overriding public function. This macro is called just once even if the function name is overloaded and the same override type can be used for all functions in the same overload set (see Advanced Topics).
