Macro BOOST_CONTRACT_DYN_LINK

BOOST_CONTRACT_DYN_LINK — Automatically defined when this library is being compiled as a shared library (defined by default).

Synopsis

// In header: <boost/contract/core/config.hpp>

BOOST_CONTRACT_DYN_LINK

Description

This library will define this macro unless users define BOOST_CONTRACT_STATIC_LINK or BOOST_CONTRACT_HEADER_ONLY (this macro is not a configuration macro and this library will generate a compile-time error if users try to define it directly). Programmers should rarely, if ever, need to use this macro directly.

Warning

In general this library will correctly check contracts at run-time only when compiled as a shared library (i.e., this macro will be automatically defined by default), unless user code checks contracts in a single program unit (e.g., a single program with only statically linked libraries that check contracts).
