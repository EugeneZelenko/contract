Function function

boost::contract::function — Program contracts for non-member, private and protected functions.

Synopsis

// In header: <boost/contract/function.hpp>


specify_precondition_old_postcondition function();

Description

This is used to specify preconditions, postconditions, and old value assignments at body for non-member, private and protected functions (these functions never check class invariants, see Contract Programming Overview).

For optimization, this can be omitted for functions that do not have preconditions and postconditions.

See Also:

Tutorial

Returns:

The result of this function must be assigned to a variable of type boost::contract::guard declared locally just before the body of the contracted function (otherwise this library will generate a run-time error, see BOOST_CONTRACT_ON_MISSING_GUARD).
