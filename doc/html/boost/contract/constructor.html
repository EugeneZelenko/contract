Function template constructor

boost::contract::constructor — Program contracts for constructors.

Synopsis

// In header: <boost/contract/constructor.hpp>


template<typename Class> specify_old_postcondition constructor(Class * obj);

Description

This is used to specify postconditions, old value assignments at body, and check class invariants for constructors (see boost::contract::constructor_precondition to specify preconditions for constructors instead).

For optimization, this can be omitted for constructors that do not have postconditions when the enclosing class has no invariants.

See Also:

Tutorial

Parameters:

obj

The object this from the scope of the contracted constructor.

Returns:

The result of this function must be assigned to a variable of type boost::contract::guard declared locally just before the body of the contracted constructor (otherwise this library will generate a run-time error, see BOOST_CONTRACT_ON_MISSING_GUARD).
