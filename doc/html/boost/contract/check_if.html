Function template check_if

boost::contract::check_if — Select compilation and execution of a boolean functor check via a static boolean nullary meta-function.

Synopsis

// In header: <boost/contract/call_if.hpp>


template<typename Cond, typename Check> bool check_if(Check f);

Description

Compile and execute a boolean nullary functor call if and only if the specified static condition is true, otherwise trivially return true.

See Also:

Advanced Topics

Parameters:

f

Boolean nullary templated functor. The functor call f() is compiled and executed if and only if Cond::value is true.

Template Parameters:

Cond

Static boolean nullary meta-function selecting when the functor call should be compiled and executed.

Returns:

Boolean value returned by f() if the static condition if true, otherwise simply return true (i.e., check trivially passed).
