Macro BOOST_CONTRACT_NO_FUNCTIONS

BOOST_CONTRACT_NO_FUNCTIONS — Defined by this library if non-member, private, and protected function contracts are not being checked by boost::contract::function.

Synopsis

// In header: <boost/contract/core/config.hpp>

BOOST_CONTRACT_NO_FUNCTIONS

Description

Users can manually program #ifdef statements in their code using this macro to completely disable compilation of non-member, private, and protected function contracts (not recommended).

This macro is automatically defined by this library, and it is not a configuration macro. This library will generate a compile-time error if programmers try to manually define this macro. Programmers can manually define BOOST_CONTRACT_NO_PRECONDITIONS and BOOST_CONTRACT_NO_POSTCONDITIONS instead (private and protected functions do not check class invariants, see also Contract Programming Overview).
